# Sobre mim

Olá, me chamo Lucas Daher e sou fundador da Codewynn, uma empresa de tecnologia focada em construir estrutura para escala empresarial por meio de tecnologia e inteligência artificial.

No meu dia a dia, atuo unindo visão de negócio e execução técnica. Analiso processos, identifico gargalos, estruturo soluções digitais e lidero o desenvolvimento de produtos que ajudam empresas a ganhar mais controle, velocidade e eficiência operacional.

Na Codewynn, nosso foco é aplicar tecnologia de forma prática para resolver problemas reais, criando soluções sob medida que entregam valor, performance e clareza para os clientes e possibilitando segurança e estrutura para crescimento.
